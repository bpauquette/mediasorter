# Refund Policy (Template)

Date: March 6, 2026

This template should be reviewed by your attorney before production use.

## Installer Purchase Refunds

If a customer cannot install or run MediaSorter on a supported Windows version, they may request a refund within 7 calendar days of purchase.

## What We Need

To process a refund request quickly, ask for:

- order number
- Windows version
- short description of the issue
- screenshot/error message if available

## Exclusions (Example)

- Requests after the refund window
- Issues caused by unsupported environments where requirements were clearly disclosed

## Support First

Before refunding, offer troubleshooting steps. If unresolved, issue refund promptly under this policy.
