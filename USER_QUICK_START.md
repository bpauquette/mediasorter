# User Quick Start (Windows)

## 1) Install

1. Download `MediaSorterSetup.exe`.
2. Run installer.
3. Launch MediaSorter from Start Menu.

## 2) First Run

1. Select input folder (your original photos/videos).
2. Select output folder (organized copy destination).
3. Choose folder structure.
4. Optionally enable:
   - video conversion
   - face identification after this run (pass one)
   - trial mode
5. Click `Start Sorting`.
6. Use `Show Advanced Controls` only if you need manual tuning.

## 3) Trial Mode

- `Try Before You Buy` processes the first N items.
- Default trial limit is 200 items.

## 4) After Sorting

- Review output folders.
- For later passes, use `Run Face Identification On Existing Output (Later Pass)`.
- Use `Support / Buy` to unlock full-library runs after trial.
- Use `Open App Data Folder` (Advanced) for logs and local settings files.

## 5) Troubleshooting

- HEIC not loading: install runtime with HEIC support enabled.
- Video conversion needs HandBrake installed.
- If face scan unavailable, ensure OpenCV face modules are present in the runtime.
